# Спецификация языка MiniLang

## **Обзор**
MiniLang — это упрощенный C-подобный язык программирования, разработанный для учебного компилятора. Данный документ описывает лексическую, синтаксическую и семантическую структуру языка.

---

## **1. Алфавит и кодировка**

### **1.1 Набор символов**
- **Буквы:** `a-z`, `A-Z`
- **Цифры:** `0-9`
- **Специальные символы:** `_` (подчеркивание)
- **Пробельные символы:** пробел (` `), табуляция (`\t`), новая строка (`\n`), возврат каретки (`\r`)
- **Кодировка:** UTF-8 (поддерживаются только ASCII-символы, не-ASCII символы игнорируются)

---

## **2. Лексемы**

### **2.1 Ключевые слова (Keywords)**
Ключевые слова зарезервированы и не могут использоваться как идентификаторы.

```
fn          // Объявление функции
if          // Условный оператор
else        // Альтернативная ветка
while       // Цикл с предусловием
for         // Цикл со счетчиком
return      // Возврат из функции

int         // Целочисленный тип (32-bit)
float       // Число с плавающей точкой
bool        // Логический тип
string      // Строковый тип

true        // Логическая истина
false       // Логическая ложь
struct      // Структура (составной тип)
const       // Константа
```

### **2.2 Идентификаторы**
```
Идентификатор ::= Буква (Буква | Цифра | '_')*

Буква        ::= [a-zA-Z]
Цифра        ::= [0-9]
```

**Правила:**
- Должны начинаться с буквы
- Могут содержать буквы, цифры и подчеркивания
- Чувствительны к регистру (`var` ≠ `Var` ≠ `VAR`)
- Максимальная длина: 255 символов
- Не могут совпадать с ключевыми словами

**Примеры:**
```
x
counter
_temp
var123
very_long_identifier_name
```

### **2.3 Литералы**

#### **2.3.1 Целочисленные литералы**
```
ЦелыйЛитерал ::= Цифра+
```

- Только десятичная система счисления
- Диапазон значений: `[-2³¹, 2³¹-1]` (от -2147483648 до 2147483647)
- Поддержка отрицательных чисел (через унарный оператор `-`)

**Примеры:**
```
0
42
-100
2147483647
```

#### **2.3.2 Литералы с плавающей точкой**
```
ВещественныйЛитерал ::= Цифра+ '.' Цифра+ [ ('e'|'E') ['+'|'-'] Цифра+ ]
```

**Примеры:**
```
3.14
0.5
2.0
1.5e-3      // Научная нотация: 0.0015
2.5E+6      // Научная нотация: 2500000.0
```

#### **2.3.3 Строковые литералы**
```
СтроковыйЛитерал ::= '"' (Символ | EscapeПоследовательность)* '"'

EscapeПоследовательность ::= '\\' ('n' | 't' | 'r' | 'b' | 'f' | 'v' | '0' | '"' | '\\' | '\'')
```

**Поддерживаемые escape-последовательности:**
| Последовательность | Значение | Описание |
|-------------------|----------|----------|
| `\n` | 0x0A | Новая строка |
| `\t` | 0x09 | Табуляция |
| `\r` | 0x0D | Возврат каретки |
| `\b` | 0x08 | Backspace |
| `\f` | 0x0C | Form feed |
| `\v` | 0x0B | Вертикальная табуляция |
| `\0` | 0x00 | Нулевой символ |
| `\"` | 0x22 | Двойная кавычка |
| `\\` | 0x5C | Обратный слеш |
| `\'` | 0x27 | Одинарная кавычка |

**Примеры:**
```
"Hello, World!"
"Line 1\nLine 2"
"Path: C:\\Users\\test"
"He said: \"Hello!\""
```

#### **2.3.4 Логические литералы**
```
ЛогическийЛитерал ::= 'true' | 'false'
```

### **2.4 Операторы**

#### **2.4.1 Арифметические операторы**
| Оператор | Значение | Пример |
|----------|----------|---------|
| `+` | Сложение | `a + b` |
| `-` | Вычитание | `a - b` |
| `*` | Умножение | `a * b` |
| `/` | Деление | `a / b` |
| `%` | Остаток от деления | `a % b` |

#### **2.4.2 Инкремент/декремент**
| Оператор | Значение | Пример |
|----------|----------|---------|
| `++` | Инкремент | `x++` или `++x` |
| `--` | Декремент | `x--` или `--x` |

#### **2.4.3 Операторы сравнения**
| Оператор | Значение | Пример |
|----------|----------|---------|
| `==` | Равно | `a == b` |
| `!=` | Не равно | `a != b` |
| `<` | Меньше | `a < b` |
| `<=` | Меньше или равно | `a <= b` |
| `>` | Больше | `a > b` |
| `>=` | Больше или равно | `a >= b` |

#### **2.4.4 Логические операторы**
| Оператор | Значение | Пример |
|----------|----------|---------|
| `&&` | Логическое И | `a && b` |
| `\|\|` | Логическое ИЛИ | `a \|\| b` |
| `!` | Логическое НЕ | `!a` |

#### **2.4.5 Операторы присваивания**
| Оператор | Значение | Пример | Эквивалент |
|----------|----------|---------|------------|
| `=` | Присваивание | `x = 5` | `x = 5` |
| `+=` | Сложение с присваиванием | `x += 3` | `x = x + 3` |
| `-=` | Вычитание с присваиванием | `x -= 3` | `x = x - 3` |
| `*=` | Умножение с присваиванием | `x *= 3` | `x = x * 3` |
| `/=` | Деление с присваиванием | `x /= 3` | `x = x / 3` |
| `%=` | Остаток с присваиванием | `x %= 3` | `x = x % 3` |

### **2.5 Разделители**

| Символ | Назначение |
|--------|------------|
| `(` | Начало списка параметров/выражения |
| `)` | Конец списка параметров/выражения |
| `{` | Начало блока кода |
| `}` | Конец блока кода |
| `[` | Начало индекса массива |
| `]` | Конец индекса массива |
| `;` | Разделитель инструкций |
| `,` | Разделитель элементов в списке |
| `.` | Доступ к полям структуры |
| `:` | Метка |
| `->` | Указание возвращаемого типа функции |

## **5. Грамматика языка (EBNF)**

### **5.1 Программа**
```
Program ::= { Declaration }
```

### **5.2 Объявления**
```
Declaration ::= FunctionDecl
              | StructDecl

FunctionDecl ::= 'fn' Identifier '(' [ Parameters ] ')' [ '->' Type ] Block

StructDecl   ::= 'struct' Identifier '{' { VarDecl } '}'

VarDecl      ::= Type Identifier [ '=' Expression ] ';'

Parameters   ::= Parameter { ',' Parameter }
Parameter    ::= Type Identifier

Type         ::= 'int' | 'float' | 'bool' | 'string' | Identifier   (типы структур)
```

### **5.3 Инструкции**
```
Statement    ::= Block
               | IfStmt
               | WhileStmt
               | ForStmt
               | ReturnStmt
               | ExprStmt
               | VarDecl               (локальные переменные)
               | EmptyStmt

Block        ::= '{' { Statement } '}'

IfStmt       ::= 'if' '(' Expression ')' Statement [ 'else' Statement ]

WhileStmt    ::= 'while' '(' Expression ')' Statement

ForStmt      ::= 'for' '(' [ ExprStmt ] ';' [ Expression ] ';' [ ExprStmt ] ')' Statement

ReturnStmt   ::= 'return' [ Expression ] ';'

ExprStmt     ::= Expression ';'

EmptyStmt    ::= ';'
```

### **5.4 Выражения (с приоритетом операторов)**
```
Expression   ::= Assignment

Assignment   ::= LogicalOr [ ('=' | '+=' | '-=' | '*=' | '/=' | '%=') Assignment ]

LogicalOr    ::= LogicalAnd { '||' LogicalAnd }

LogicalAnd   ::= Equality { '&&' Equality }

Equality     ::= Relational { ('==' | '!=') Relational }

Relational   ::= Additive { ('<' | '<=' | '>' | '>=') Additive }

Additive     ::= Multiplicative { ('+' | '-') Multiplicative }

Multiplicative ::= Unary { ('*' | '/' | '%') Unary }

Unary        ::= ('-' | '!' | '++' | '--') Unary
               | Postfix

Postfix      ::= Primary { PostfixOp }

PostfixOp    ::= '++'                // постфиксный инкремент
               | '--'                // постфиксный декремент
               | CallSuffix           // вызов функции
               | IndexSuffix          // индексация массива
               | MemberAccess         // доступ к полю структуры

CallSuffix   ::= '(' [ Arguments ] ')'

IndexSuffix  ::= '[' Expression ']'

MemberAccess ::= '.' Identifier

Primary      ::= Literal
               | Identifier
               | '(' Expression ')'
               | Call

Call         ::= Identifier '(' [ Arguments ] ')'

Arguments    ::= Expression { ',' Expression }

Literal      ::= IntegerLiteral | FloatLiteral | StringLiteral | BooleanLiteral
```

## **8. Семантические правила**

### **8.1 Области видимости**
| Область | Уровень | Содержимое |
|---------|---------|------------|
| Глобальная | 0 | Функции, структуры |
| Функциональная | 1 | Параметры, локальные переменные |
| Блочная | 2+ | Переменные внутри блоков `{ }` |
| Структурная | - | Поля структур |

### **8.2 Система типов**

#### **Базовые типы**
| Тип | Размер (байт) | Выравнивание | Описание |
|-----|---------------|--------------|----------|
| `int` | 4 | 4 | 32-битное целое число |
| `float` | 4 | 4 | 32-битное число с плавающей точкой |
| `bool` | 1 | 1 | Логическое значение (true/false) |
| `string` | 8 | 8 | Указатель на строку (64-bit) |
| `void` | 0 | 0 | Пустой тип (только для функций) |

#### **Составные типы**
| Тип | Описание | Пример |
|-----|----------|---------|
| Структура | Пользовательский тип с полями | `struct Point { int x; int y; }` |
| Массив | Последовательность элементов одного типа | `int arr[10]` |

### **8.3 Неявные преобразования типов**
| Преобразование | Разрешено | Пример |
|----------------|-----------|---------|
| `int` → `float` | ✅ | `float x = 5;` |
| `float` → `int` | ❌ | `int x = 3.14;` (ошибка) |
| `bool` → `int` | ❌ | `int x = true;` (ошибка) |
| `string` → `int` | ❌ | `int x = "hello";` (ошибка) |

### **8.4 Правила для операторов**

#### **Арифметические операторы (`+ - * / %`)**
| Левый тип | Правый тип | Результат |
|-----------|------------|-----------|
| int | int | int |
| int | float | float |
| float | int | float |
| float | float | float |

#### **Операторы сравнения (`== != < <= > >=`)**
- Оба операнда должны быть одного типа (или совместимы)
- Результат: `bool`

#### **Логические операторы (`&& || !`)**
- Операнды должны быть типа `bool`
- Результат: `bool`

#### **Унарный минус (`-`)**
- Операнд должен быть числовым (`int` или `float`)
- Результат: тот же тип, что и операнд

#### **Логическое НЕ (`!`)**
- Операнд должен быть типа `bool`
- Результат: `bool`

### **8.5 Правила для функций**

| Проверка | Описание |
|----------|----------|
| Количество аргументов | Должно совпадать с количеством параметров |
| Типы аргументов | Должны быть совместимы с типами параметров |
| Тип возврата | Должен соответствовать объявлению функции |
| Оператор `return` | Обязателен для функций с типом возврата (кроме `void`) |

### **8.6 Правила для переменных**

| Проверка | Описание |
|----------|----------|
| Объявление | Переменная должна быть объявлена перед использованием |
| Дублирование | Нельзя объявить переменную дважды в одной области видимости |
| Область видимости | Переменная доступна только внутри своей области |

### **8.7 Правила для структур**

| Проверка | Описание |
|----------|----------|
| Поля | Имена полей в структуре должны быть уникальны |
| Доступ | Доступ к полям осуществляется через оператор `.` |
| Существование | Поле должно существовать в структуре |

### **8.8 Правила для массивов**

| Проверка | Описание |
|----------|----------|
| Индексация | Оператор `[]` можно использовать только с массивами |
| Тип индекса | Индекс должен быть целым числом (`int`) |

## **9. Сообщения об ошибках**

### **9.1 Формат ошибок**
```
ОШИБКА [КОД]: Сообщение
Файл:<filename>
Строка:<line>, позиция:<column>
Код:<line-content>
<position-indicator>
Подсказка:<suggestion>
```

### **9.2 Коды ошибок**

#### **Лексические ошибки (LEX-xxx)**
| Код | Сообщение | Подсказка |
|-----|-----------|-----------|
| LEX-001 | Недопустимый символ: '%c' | Удалите или замените недопустимый символ |
| LEX-002 | Неверное целое число: '%s' | Используйте только цифры для целых чисел |
| LEX-004 | Незакрытая строка | Добавьте закрывающую кавычку " |
| LEX-005 | Символ новой строки внутри строки | Используйте \n для переноса строки |
| LEX-006 | Неверная escape-последовательность: \%c | Используйте \n \t \r \" \\ |
| LEX-008 | Слишком длинный идентификатор | Сократите имя до 255 символов |

#### **Синтаксические ошибки (SYN-xxx)**
| Код | Сообщение | Подсказка |
|-----|-----------|-----------|
| SYN-001 | Пропущена точка с запятой перед '%s' | Добавьте ';' в конце оператора |
| SYN-002 | Пропущена закрывающая скобка ')' | Добавьте ')' |
| SYN-003 | Незакрытый блок (ожидается '}') | Добавьте '}' |
| SYN-004 | Неожиданный токен '%s' | Проверьте синтаксис вокруг этого токена |
| SYN-005 | Ожидался идентификатор, получен '%s' | Укажите имя переменной или функции |

#### **Семантические ошибки (SEM-xxx)**
| Код | Сообщение | Подсказка |
|-----|-----------|-----------|
| SEM-001 | Необъявленный идентификатор: '%s' | Объявите переменную перед использованием |
| SEM-002 | Повторное объявление: '%s' | Используйте другое имя |
| SEM-003 | Несоответствие типов: ожидался '%s', получен '%s' | Проверьте соответствие типов |
| SEM-005 | Неверное количество аргументов функции '%s' | Проверьте количество аргументов |
| SEM-007 | Несовместимый тип возврата | Функция должна возвращать указанный тип |
| SEM-009 | Условие должно иметь тип bool | Используйте логическое выражение |

## **10. Примеры кода**

### **10.1 Простая программа**
```c
fn main() -> int {
    int x = 42;
    float y = 3.14;
    string msg = "Hello, World!";
    bool flag = true;
    
    if (x > 0 && flag) {
        x = x + 1;
    }
    
    return 0;
}
```

### **10.2 Функции и арифметика**
```c
fn add(int a, int b) -> int {
    return a + b;
}

fn main() -> int {
    int result = add(5, 3);
    return result;
}
```

### **10.3 Структуры и доступ к полям**
```c
struct Point {
    int x;
    int y;
}

fn main() -> int {
    Point p;
    p.x = 10;
    p.y = 20;
    return p.x + p.y;
}
```

### **10.4 Массивы**
```c
fn sumArray(int arr[], int size) -> int {
    int sum = 0;
    for (int i = 0; i < size; i = i + 1) {
        sum = sum + arr[i];
    }
    return sum;
}
```

## **11. Ограничения реализации**

- Максимальная длина идентификатора: 255 символов
- Максимальный размер исходного файла: не ограничен (тестировалось до 1 МБ)
- Максимальная глубина вложенности комментариев: не ограничена
- Максимальная глубина вложенности условных директив: не ограничена
- Поддержка Unicode: только ASCII-символы
