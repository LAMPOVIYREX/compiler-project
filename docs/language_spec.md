# Спецификация языка MiniLang

## **Обзор**
MiniLang — это упрощенный C-подобный язык программирования, разработанный для учебного компилятора. Данный документ описывает лексическую и синтаксическую структуру языка.

---

## **1. Алфавит и кодировка**

### **1.1 Набор символов**
- **Буквы:** `a-z`, `A-Z`
- **Цифры:** `0-9`
- **Специальные символы:** `_` (подчеркивание)
- **Пробельные символы:** пробел (` `), табуляция (`\t`), новая строка (`\n`), возврат каретки (`\r`)
- **Кодировка:** UTF-8 (поддерживаются только ASCII-символы, не-ASCII символы игнорируются)

## **2. Лексемы**

### **2.1 Ключевые слова Keywords**
Ключевые слова зарезервированы и не могут использоваться как идентификаторы.

```
fn          // Объявление функции
if          // Условный оператор
else        // Альтернативная ветка
while       // Цикл с предусловием
for         // Цикл со счетчиком
return      // Возврат из функции

int         // Целочисленный тип (32-bit)
float       // Число с плавающей точкой
bool        // Логический тип
string      // Строковый тип

true        // Логическая истина
false       // Логическая ложь
struct      // Структура (составной тип)
const       // Константа
```

### **2.2 Идентификаторы **
```
Идентификатор ::= Буква (Буква | Цифра | '_')*

Буква        ::= [a-zA-Z]
Цифра        ::= [0-9]
```

**Правила:**
- Должны начинаться с буквы
- Могут содержать буквы, цифры и подчеркивания
- Чувствительны к регистру (`var` ≠ `Var` ≠ `VAR`)
- Максимальная длина: 255 символов
- Не могут совпадать с ключевыми словами

**Примеры:**
```
x
counter
_temp
var123
very_long_identifier_name
```

### **2.3 Литералы **

#### **2.3.1 Целочисленные литералы**
```
ЦелыйЛитерал ::= Цифра+
```

- Только десятичная система счисления
- Диапазон значений: `[-2³¹, 2³¹-1]` (от -2147483648 до 2147483647)
- Поддержка отрицательных чисел (через унарный оператор `-`)

**Примеры:**
```
0
42
-100
2147483647
```

#### **2.3.2 Литералы с плавающей точкой**
```
ВещественныйЛитерал ::= Цифра+ '.' Цифра+ [ ('e'|'E') ['+'|'-'] Цифра+ ]
```

**Примеры:**
```
3.14
0.5
2.0
1.5e-3      // : 0.0015
2.5E+6      // : 2500000.0
```

#### **2.3.3 Строковые литералы**
```
СтроковыйЛитерал ::= '"' (Символ | EscapeПоследовательность)* '"'

EscapeПоследовательность ::= '\\' ('n' | 't' | 'r' | 'b' | 'f' | 'v' | '0' | '"' | '\\' | '\'')
```

**Поддерживаемые escape-последовательности:**
| Последовательность | Значение | Описание |
|-------------------|----------|----------|
| `\n` | 0x0A | Новая строка |
| `\t` | 0x09 | Табуляция |
| `\r` | 0x0D | Возврат каретки |
| `\b` | 0x08 | Backspace |
| `\f` | 0x0C | Form feed |
| `\v` | 0x0B | Вертикальная табуляция |
| `\0` | 0x00 | Нулевой символ |
| `\"` | 0x22 | Двойная кавычка |
| `\\` | 0x5C | Обратный слеш |
| `\'` | 0x27 | Одинарная кавычка |

**Примеры:**
```
"Hello, World!"
"Line 1\nLine 2"
"Path: C:\\Users\\test"
"He said: \"Hello!\""
```

#### **2.3.4 Логические литералы**
```
ЛогическийЛитерал ::= 'true' | 'false'
```

### **2.4 Операторы**

#### **2.4.1 Арифметические операторы**
| Оператор | Значение | Пример |
|----------|----------|---------|
| `+` | Сложение | `a + b` |
| `-` | Вычитание | `a - b` |
| `*` | Умножение | `a * b` |
| `/` | Деление | `a / b` |
| `%` | Остаток от деления | `a % b` |

#### **2.4.2 Инкремент/декремент**
| Оператор | Значение | Пример |
|----------|----------|---------|
| `++` | Инкремент | `x++` или `++x` |
| `--` | Декремент | `x--` или `--x` |

#### **2.4.3 Операторы сравнения**
| Оператор | Значение | Пример |
|----------|----------|---------|
| `==` | Равно | `a == b` |
| `!=` | Не равно | `a != b` |
| `<` | Меньше | `a < b` |
| `<=` | Меньше или равно | `a <= b` |
| `>` | Больше | `a > b` |
| `>=` | Больше или равно | `a >= b` |

#### **2.4.4 Логические операторы**
| Оператор | Значение | Пример |
|----------|----------|---------|
| `&&` | Логическое И | `a && b` |
| `\|\|` | Логическое ИЛИ | `a \|\| b` |
| `!` | Логическое НЕ | `!a` |

#### **2.4.5 Операторы присваивания**
| Оператор | Значение | Пример | Эквивалент |
|----------|----------|---------|------------|
| `=` | Присваивание | `x = 5` | `x = 5` |
| `+=` | Сложение с присваиванием | `x += 3` | `x = x + 3` |
| `-=` | Вычитание с присваиванием | `x -= 3` | `x = x - 3` |
| `*=` | Умножение с присваиванием | `x *= 3` | `x = x * 3` |
| `/=` | Деление с присваиванием | `x /= 3` | `x = x / 3` |
| `%=` | Остаток с присваиванием | `x %= 3` | `x = x % 3` |

### **2.5 Разделители**

| Символ | Назначение |
|--------|------------|
| `(` | Начало списка параметров/выражения |
| `)` | Конец списка параметров/выражения |
| `{` | Начало блока кода |
| `}` | Конец блока кода |
| `[` | Начало индекса массива |
| `]` | Конец индекса массива |
| `;` | Разделитель инструкций |
| `,` | Разделитель элементов в списке |
| `.` | Доступ к полям структуры |
| `:` | Метка или тернарный оператор |
| `->` | Указание возвращаемого типа функции |

## **5. Грамматика языка (EBNF)**

### **5.1 Программа**
```
Program ::= { Declaration }
```

### **5.2 Объявления**
```
Declaration ::= FunctionDecl
              | StructDecl

FunctionDecl ::= 'fn' Identifier '(' [ Parameters ] ')' [ '->' Type ] Block

StructDecl   ::= 'struct' Identifier '{' { VarDecl } '}'

VarDecl      ::= Type Identifier [ '=' Expression ] ';'

Parameters   ::= Parameter { ',' Parameter }
Parameter    ::= Type Identifier

Type         ::= 'int' | 'float' | 'bool' | 'string' | Identifier   (типы структур)
```

### **5.3 Инструкции**
```
Statement    ::= Block
               | IfStmt
               | WhileStmt
               | ForStmt
               | ReturnStmt
               | ExprStmt
               | VarDecl               (локальные переменные)
               | EmptyStmt

Block        ::= '{' { Statement } '}'

IfStmt       ::= 'if' '(' Expression ')' Statement [ 'else' Statement ]

WhileStmt    ::= 'while' '(' Expression ')' Statement

ForStmt      ::= 'for' '(' [ ExprStmt ] ';' [ Expression ] ';' [ ExprStmt ] ')' Statement

ReturnStmt   ::= 'return' [ Expression ] ';'

ExprStmt     ::= Expression ';'

EmptyStmt    ::= ';'
```

### **5.4 Выражения (с приоритетом операторов)**
```
Expression   ::= Assignment

Assignment   ::= LogicalOr [ ('=' | '+=' | '-=' | '*=' | '/=' | '%=') Assignment ]

LogicalOr    ::= LogicalAnd { '||' LogicalAnd }

LogicalAnd   ::= Equality { '&&' Equality }

Equality     ::= Relational { ('==' | '!=') Relational }

Relational   ::= Additive { ('<' | '<=' | '>' | '>=') Additive }

Additive     ::= Multiplicative { ('+' | '-') Multiplicative }

Multiplicative ::= Unary { ('*' | '/' | '%') Unary }

Unary        ::= ('-' | '!' | '++' | '--') Unary
               | Postfix

Postfix      ::= Primary { PostfixOp }

PostfixOp    ::= '++'                // постфиксный инкремент
               | '--'                // постфиксный декремент
               | CallSuffix           // вызов функции
               | IndexSuffix          // индексация массива
               | MemberAccess         // доступ к полю структуры

CallSuffix   ::= '(' [ Arguments ] ')'

IndexSuffix  ::= '[' Expression ']'

MemberAccess ::= '.' Identifier

Primary      ::= Literal
               | Identifier
               | '(' Expression ')'
               | Call

Call         ::= Identifier '(' [ Arguments ] ')'

Arguments    ::= Expression { ',' Expression }

Literal      ::= IntegerLiteral | FloatLiteral | StringLiteral | BooleanLiteral
```

## **6. Приоритет и ассоциативность операторов**

| Уровень | Операторы | Ассоциативность |

| 7 | `==` `!=` | Левая |
| 8 | `&&` | Левая |
| 9 | `\|\|` | Левая |
| 10 | `=` `+=` `-=` `*=` `/=` `%=` | Правая |

## **8. Примеры кода**

### **8.1 Простая программа**
```c
fn main() -> int {
    int x = 42;
    float y = 3.14;
    string msg = "Hello, World!";
    bool flag = true;
    
    if (x > 0 && flag) {
        x = x + 1;
    }
    
    return 0;
}
```

### **8.2 Функции и арифметика**
```c
fn add(int a, int b) -> int {
    return a + b;
}

fn main() -> int {
    int result = add(5, 3);
    return result;
}
```

### **8.3 Рекурсивный факториал**
```c
fn factorial(int n) -> int {
    if (n <= 1) {
        return 1;
    }
    return n * factorial(n - 1);
}

fn main() -> int {
    return factorial(5);
}
```

### **8.4 Циклы**
```c
fn sum(int n) -> int {
    int result = 0;
    int i = 1;
    
    while (i <= n) {
        result = result + i;
        i = i + 1;
    }
    
    return result;
}

fn sumFor(int n) -> int {
    int result = 0;
    
    for (int i = 1; i <= n; i = i + 1) {
        result = result + i;
    }
    
    return result;
}
```

### **8.5 Массивы**
```c
fn sumArray(int arr[], int size) -> int {
    int sum = 0;
    
    for (int i = 0; i < size; i = i + 1) {
        sum = sum + arr[i];
    }
    
    return sum;
}
```

### **8.6 Структуры**
```c
struct Point {
    int x;
    int y;
}

struct Rectangle {
    Point topLeft;
    Point bottomRight;
}

fn main() -> int {
    Point p;
    p.x = 10;
    p.y = 20;
    
    Rectangle r;
    r.topLeft = p;
    r.bottomRight.x = 100;
    r.bottomRight.y = 200;
    
    return 0;
}
```

### **8.7 Использование препроцессора**
```c
#define ARRAY_SIZE 100
#define PI 3.14159
#define DEBUG 1

fn calculate() {
    int arr[ARRAY_SIZE];
    float area = PI * 5 * 5;
    
    #ifdef DEBUG
        // Отладочный код
    #endif
    
    #if ARRAY_SIZE > 50
        // Работа с большим массивом
    #endif
    
    return 0;
}
```

## **9. Сообщения об ошибках**

### **9.1 Формат ошибок**
```
[ERROR] <filename>:<line>:<column>: <error-code>: <error-message>
[CONTEXT] <line-content>
[POINTER] <position-indicator>
```

### **9.2 Коды ошибок**

#### **Лексические ошибки (LEX-xxx)**
| Код | Сообщение |
|-----|-----------|
| LEX-001 | `Unexpected character: '%c' (ASCII: %d)` |
| LEX-002 | `Invalid integer literal: '%s'` |
| LEX-003 | `Invalid float literal: '%s'` |
| LEX-004 | `Unterminated string literal` |
| LEX-005 | `Newline in string literal (use \n)` |
| LEX-006 | `Invalid escape sequence: \%c` |
| LEX-007 | `Unterminated multi-line comment` |
| LEX-008 | `Identifier too long` |

#### **Синтаксические ошибки (SYN-xxx)**
| Код | Сообщение |
|-----|-----------|
| SYN-001 | `Expected '%s' but found '%s'` |
| SYN-002 | `Unexpected token '%s'` |
| SYN-003 | `Missing '%s' after %s` |
| SYN-004 | `Invalid expression` |
| SYN-005 | `Expected identifier` |
| SYN-006 | `Unclosed block` |

#### **Семантические ошибки (SEM-xxx)**
| Код | Сообщение |
|-----|-----------|
| SEM-001 | `Undeclared identifier: '%s'` |
| SEM-002 | `Redeclaration of identifier: '%s'` |
| SEM-003 | `Type mismatch` |
| SEM-004 | `Function '%s' not declared` |
| SEM-005 | `Wrong number of arguments` |

## **10. Ограничения реализации**

- Максимальная длина идентификатора: 255 символов
- Максимальный размер исходного файла: не ограничен (тестировалось до 1 МБ)
- Максимальная глубина вложенности комментариев: не ограничена
- Максимальная глубина вложенности условных директив: не ограничена
- Поддержка Unicode: только ASCII-символы
