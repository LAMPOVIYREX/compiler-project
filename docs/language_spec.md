# **Спецификация языка MiniLang**

## **Обзор**
MiniLang — это упрощенный C-подобный язык программирования, разработанный для учебного компилятора. Данный документ описывает лексическую структуру языка.

---

## **1. Алфавит и кодировка**

### **1.1 Набор символов**
- **Буквы:** `a-z`, `A-Z`
- **Цифры:** `0-9`
- **Специальные символы:** `_` (подчеркивание)
- **Пробельные символы:** пробел (` `), табуляция (`\t`), новая строка (`\n`), возврат каретки (`\r`)
- **Кодировка:** UTF-8 (поддерживаются только ASCII-символы, не-ASCII символы игнорируются)

---

## **2. Лексемы**

### **2.1 Ключевые слова (Keywords)**
Ключевые слова зарезервированы и не могут использоваться как идентификаторы.

```
fn          // Объявление функции
if          // Условный оператор
else        // Альтернативная ветка
while       // Цикл с предусловием
for         // Цикл со счетчиком
return      // Возврат из функции

int         // Целочисленный тип (32-bit)
float       // Число с плавающей точкой
bool        // Логический тип
void        // Пустой тип
string      // Строковый тип

true        // Логическая истина
false       // Логическая ложь
struct      // Структура (составной тип)
const       // Константа
```

### **2.2 Идентификаторы (Identifiers)**
```
Идентификатор ::= Буква (Буква | Цифра | '_')*

Буква        ::= [a-zA-Z]
Цифра        ::= [0-9]
```

**Правила:**
- Должны начинаться с буквы
- Могут содержать буквы, цифры и подчеркивания
- Чувствительны к регистру (`var` ≠ `Var` ≠ `VAR`)
- Максимальная длина: 255 символов
- Не могут совпадать с ключевыми словами

**Примеры:**
```
x
counter
_temp
var123
very_long_identifier_name
```

### **2.3 Литералы (Literals)**

#### **2.3.1 Целочисленные литералы**
```
ЦелыйЛитерал ::= Цифра+
```

- Только десятичная система счисления
- Диапазон значений: `[-2³¹, 2³¹-1]` (от -2147483648 до 2147483647)
- Поддержка отрицательных чисел (через унарный оператор `-`)

**Примеры:**
```
0
42
-100
2147483647
```

#### **2.3.2 Литералы с плавающей точкой**
```
ВещественныйЛитерал ::= Цифра+ '.' Цифра+ [ ('e'|'E') ['+'|'-'] Цифра+ ]
```

**Примеры:**
```
3.14
0.5
2.0
1.5e-3      // Научная нотация: 0.0015
2.5E+6      // Научная нотация: 2500000.0
```

#### **2.3.3 Строковые литералы**
```
СтроковыйЛитерал ::= '"' (Символ | EscapeПоследовательность)* '"'

EscapeПоследовательность ::= '\\' ('n' | 't' | 'r' | 'b' | 'f' | 'v' | '0' | '"' | '\\' | '\'')
```

**Поддерживаемые escape-последовательности:**
| Последовательность | Значение | Описание |
|-------------------|----------|----------|
| `\n` | 0x0A | Новая строка |
| `\t` | 0x09 | Табуляция |
| `\r` | 0x0D | Возврат каретки |
| `\b` | 0x08 | Backspace |
| `\f` | 0x0C | Form feed |
| `\v` | 0x0B | Вертикальная табуляция |
| `\0` | 0x00 | Нулевой символ |
| `\"` | 0x22 | Двойная кавычка |
| `\\` | 0x5C | Обратный слеш |
| `\'` | 0x27 | Одинарная кавычка |

**Примеры:**
```
"Hello, World!"
"Line 1\nLine 2"
"Path: C:\\Users\\test"
"He said: \"Hello!\""
```

#### **2.3.4 Логические литералы**
```
ЛогическийЛитерал ::= 'true' | 'false'
```

### **2.4 Операторы (Operators)**

#### **2.4.1 Арифметические операторы**
| Оператор | Значение | Пример |
|----------|----------|---------|
| `+` | Сложение | `a + b` |
| `-` | Вычитание | `a - b` |
| `*` | Умножение | `a * b` |
| `/` | Деление | `a / b` |
| `%` | Остаток от деления | `a % b` |

#### **2.4.2 Инкремент/декремент**
| Оператор | Значение | Пример |
|----------|----------|---------|
| `++` | Инкремент | `x++` или `++x` |
| `--` | Декремент | `x--` или `--x` |

#### **2.4.3 Операторы сравнения**
| Оператор | Значение | Пример |
|----------|----------|---------|
| `==` | Равно | `a == b` |
| `!=` | Не равно | `a != b` |
| `<` | Меньше | `a < b` |
| `<=` | Меньше или равно | `a <= b` |
| `>` | Больше | `a > b` |
| `>=` | Больше или равно | `a >= b` |

#### **2.4.4 Логические операторы**
| Оператор | Значение | Пример |
|----------|----------|---------|
| `&&` | Логическое И | `a && b` |
| `\|\|` | Логическое ИЛИ | `a \|\| b` |
| `!` | Логическое НЕ | `!a` |

#### **2.4.5 Операторы присваивания**
| Оператор | Значение | Пример | Эквивалент |
|----------|----------|---------|------------|
| `=` | Присваивание | `x = 5` | `x = 5` |
| `+=` | Сложение с присваиванием | `x += 3` | `x = x + 3` |
| `-=` | Вычитание с присваиванием | `x -= 3` | `x = x - 3` |
| `*=` | Умножение с присваиванием | `x *= 3` | `x = x * 3` |
| `/=` | Деление с присваиванием | `x /= 3` | `x = x / 3` |
| `%=` | Остаток с присваиванием | `x %= 3` | `x = x % 3` |

### **2.5 Разделители (Delimiters)**

| Символ | Назначение |
|--------|------------|
| `(` | Начало списка параметров/выражения |
| `)` | Конец списка параметров/выражения |
| `{` | Начало блока кода |
| `}` | Конец блока кода |
| `[` | Начало индекса массива |
| `]` | Конец индекса массива |
| `;` | Разделитель инструкций |
| `,` | Разделитель элементов в списке |
| `.` | Доступ к полям структуры |
| `:` | Метка или тернарный оператор |

## **5. Препроцессор**

### **5.1 Директивы препроцессора**
| Директива | Описание | Пример |
|-----------|----------|---------|
| `#define` | Определение макроса | `#define MAX 100` |
| `#undef` | Удаление макроса | `#undef MAX` |
| `#ifdef` | Если макрос определен | `#ifdef DEBUG` |
| `#ifndef` | Если макрос не определен | `#ifndef VERSION` |
| `#if` | Условная компиляция | `#if MAX > 50` |
| `#elif` | Альтернативное условие | `#elif MIN < 10` |
| `#else` | Альтернативная ветка | `#else` |
| `#endif` | Конец условия | `#endif` |
| `#error` | Генерация ошибки | `#error "Message"` |
| `#warning` | Генерация предупреждения | `#warning "Deprecated"` |

### **5.2 Встроенные макросы**
| Макрос | Описание |
|--------|----------|
| `__LINE__` | Текущая строка |
| `__FILE__` | Текущий файл |
| `__VERSION__` | Версия компилятора |

### **5.3 Примеры использования препроцессора**
```c
#define ARRAY_SIZE 100
#define PI 3.14159
#define DEBUG 1

#ifdef DEBUG
    #define LOG(msg) print("Debug: " + msg)
#endif

#if ARRAY_SIZE > 50
    int largeArray[ARRAY_SIZE];
#endif
```

## **6. Примеры кода**

### **6.1 Простая программа**
```c
fn main() {
    int x = 42;
    float y = 3.14;
    string msg = "Hello, World!";
    bool flag = true;
    
    if (x > 0 && flag) {
        x = x + 1;
    }
    
    return 0;
}
```

### **6.2 Функция с параметрами**
```c
fn add(int a, int b) {
    return a + b;
}

fn main() {
    int result = add(5, 3);
    return result;
}
```

### **6.3 Циклы**
```c
fn factorial(int n) {
    int result = 1;
    
    for (int i = 1; i <= n; i++) {
        result = result * i;
    }
    
    return result;
}
```

### **6.4 Массивы**
```c
fn sumArray(int arr[], int size) {
    int sum = 0;
    
    for (int i = 0; i < size; i++) {
        sum = sum + arr[i];
    }
    
    return sum;
}
```

## **7. Грамматика в EBNF**

```
Программа        ::= { ОбъявлениеФункции }

ОбъявлениеФункции ::= 'fn' Идентификатор '(' [СписокПараметров] ')' Блок

СписокПараметров  ::= Параметр { ',' Параметр }
Параметр         ::= Тип Идентификатор

Тип              ::= 'int' | 'float' | 'bool' | 'string' | 'void'

Блок             ::= '{' { Инструкция } '}'

Инструкция       ::= ОбъявлениеПеременной ';'
                   | Присваивание ';'
                   | ВызовФункции ';'
                   | 'return' [Выражение] ';'
                   | 'if' '(' Выражение ')' Блок ['else' Блок]
                   | 'while' '(' Выражение ')' Блок
                   | 'for' '(' [Присваивание] ';' [Выражение] ';' [Присваивание] ')' Блок
                   | Блок

ОбъявлениеПеременной ::= Тип Идентификатор ['=' Выражение]

Присваивание      ::= Идентификатор ОператорПрисваивания Выражение
ОператорПрисваивания ::= '=' | '+=' | '-=' | '*=' | '/=' | '%='

Выражение         ::= ЛогическоеИЛИ
ЛогическоеИЛИ    ::= ЛогическоеИ { '||' ЛогическоеИ }
ЛогическоеИ      ::= Сравнение { '&&' Сравнение }
Сравнение        ::= Аддитивное [ ('==' | '!=' | '<' | '<=' | '>' | '>=') Аддитивное ]
Аддитивное       ::= Мультипликативное { ('+' | '-') Мультипликативное }
Мультипликативное ::= Унарное { ('*' | '/' | '%') Унарное }
Унарное          ::= ('!' | '-' | '++' | '--') Унарное | Первичное
Первичное        ::= Идентификатор
                   | Литерал
                   | '(' Выражение ')'
                   | Идентификатор '(' [СписокАргументов] ')'
                   | Идентификатор '[' Выражение ']'

СписокАргументов  ::= Выражение { ',' Выражение }

Литерал          ::= ЦелыйЛитерал | ВещественныйЛитерал | СтроковыйЛитерал | ЛогическийЛитерал
```

## **8. Сообщения об ошибках**

### **8.1 Лексические ошибки**
- "Unexpected character: X"
- "Invalid number literal"
- "Unterminated string"
- "Newline character in string literal (use \\n)"
- "Invalid escape sequence: \\X"
- "Unterminated multi-line comment"
- "Identifier too long (max 255 characters)"
- "Integer literal out of range"

### **8.2 Ошибки препроцессора**
- "Unknown preprocessor directive: #X"
- "Invalid #define syntax"
- "#else without #if"
- "#endif without #if"
- "#error: message"

## **9. Ограничения реализации**

- Максимальная длина идентификатора: 255 символов
- Максимальный размер исходного файла: не ограничен (тестировалось до 1 МБ)
- Максимальная глубина вложенности комментариев: не ограничена
- Максимальная глубина вложенности условных директив: не ограничена
- Поддержка Unicode: только ASCII-символы
